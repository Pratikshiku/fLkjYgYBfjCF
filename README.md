### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 沟通记录管理
5. 客户管理
6. 客户积分
7. 客户线索管理


**管理员功能：**
1. 登录
2. 个人中心
3. 沟通类型管理 
4. 增加类型管理
5. 线索类型管理
6. 沟通记录管理
7. 客户管理
8. 客户积分管理
9. 客户线索管理
10. 员工管理
